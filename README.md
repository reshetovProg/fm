## футбольный менеджер в консоле

### тз
Обеспечить следующий функционал:
- добавлять/удалять футболистов
- футболисты описаны: 
(имя, фамилия, рейтинг, позиция, номер, клуб)
- рейтинг, позицию, номер, клуб в последствии можно менять
- футболисты хранятся в файле .xml

- добавлять/удалять клубы
- клубы описаны: (название клуба, футболисты, рейтинг)

- турнир
- создавать турнир
- добавлять/удалять в турнир клубы
- формировать турнирную таблицу из зарегистрированных клубов
- эмулировать матч
	- кто на какой минуте кому забил гол
	- эти данные должны влиять на рейтинг игрока и клуба,
	по завершению турнира
- управление турниром будет осуществляться через консоль:
	- выбрать матч
	- сыграть матч
	- вывести сетку турнира
	- выдать призерам деньги
